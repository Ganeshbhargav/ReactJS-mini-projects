# 🔢 Click Counter

A simple and interactive React application that counts how many times a button is clicked. Built using **React + Vite**, this project helps in understanding state management and component structure.

* 🔢 Increment counter on button click
* ⚡ Instant UI updates using React state
* 🧩 Component-based architecture
* 🎯 Clean and minimal UI

## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/Ganeshbhargav/ReactJS-mini-projects.git
   ```

2. Navigate to the project folder:

   ```bash
   cd click-Counter
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start development server:

   ```bash
   npm run dev
   ```
