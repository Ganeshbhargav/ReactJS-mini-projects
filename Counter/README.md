# Counter App (React)

A simple and interactive Counter application built using React and Vite. This project demonstrates basic state management using React Hooks and component-based architecture.

## 🔗 Repository

GitHub: [https://github.com/Ganeshbhargav/ReactJS-mini-projects/tree/main/Counter](https://github.com/Ganeshbhargav/ReactJS-mini-projects/tree/main/Counter)

## 🚀 Features

* Increase counter value
* Decrease counter value
* Instant UI updates using React state
* Clean and minimal design

## 🛠️ Tech Stack

* React
* Vite
* JavaScript (ES6)
* CSS

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Ganeshbhargav/ReactJS-mini-projects.git
cd ReactJS-mini-projects/Counter
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```
http://localhost:5173
```

## 🧠 How It Works

* The `Counter` component uses the `useState` hook to store the count.
* Clicking **Increase** increments the value.
* Clicking **Decrease** decrements the value.
* React automatically updates the UI when the state changes.
