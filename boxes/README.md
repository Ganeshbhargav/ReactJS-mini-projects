## 🧠 About the Project

The **Boxes** project is a beginner-friendly React application where multiple boxes are displayed with different colors and text.

This project focuses on:

* Creating reusable components
* Passing data using props
* Styling components using CSS
* Structuring a React application

Projects like this help strengthen core React concepts such as components and UI structuring ([GeeksforGeeks][1])

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/Ganeshbhargav/ReactJS-mini-projects.git

# Navigate to the project folder
cd ReactJS-mini-projects/boxes

# Install dependencies
npm install

# Start the development server
npm start
```
