# 🚀 ReactJS Mini Projects

React JS • Vite • CSS

A collection of beginner to intermediate **ReactJS projects** built using modern tools like **Vite**.

These projects demonstrate practical implementation of **components, props, state management, and UI design** in React.

## 📌 About

This repository contains multiple mini React projects created to practice and improve **frontend development skills** using React.

By building these projects you will learn:

* Component-based architecture
* Props and state management
* Event handling in React
* UI/UX development
* Fast development using Vite

These projects are perfect for:

* Beginners learning React
* Students building portfolios
* Developers practicing frontend skills

## ▶️ Getting Started

### Clone the Repository

```bash id="s2hj21"
git clone https://github.com/Ganeshbhargav/ReactJS-mini-projects.git
```

### Navigate to a Project

```bash id="6k9t8r"
cd Notifications
```

### Install Dependencies

```bash id="m5x1u2"
npm install
```

### Run the Project

```bash id="8r0p3y"
npm run dev
```

## 📦 Build for Production
