# 🚗 Speedometer App

A dynamic React application that simulates a speedometer with controls to increase and decrease speed. Built using **React + Vite**, this project demonstrates state management and interactive UI updates.

## 📌 Features

* 🚀 Increase speed with button click
* 🛑 Apply brakes to decrease speed
* ⚡ Real-time speed updates using React state
* 🎯 Simple and interactive UI

## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/Ganeshbhargav/ReactJS-mini-projects.git
   ```

2. Navigate to the project folder:

   ```bash
   cd speedometer
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start development server:

   ```bash
   npm run dev
   ```
