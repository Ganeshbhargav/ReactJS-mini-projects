# 🚀 React Notifications App

A responsive **Notifications UI** built using React.
This project demonstrates reusable components, props, and dynamic styling.

---

## 📌 Features

- 🔁 Reusable Notification Component
- 🎨 Dynamic background colors using props
- 📱 Responsive design
- 🧩 Clean component-based structure
- ⚡ Built using Vite

---

## 🛠️ Tech Stack

- React JS
- Vite
- JavaScript (ES6)
- HTML5
- CSS3

---

## ⚙️ Installation & Setup

**Clone the repository:**

```bash
git clone https://github.com/Ganeshbhargav/ReactJS-mini-projects.git
```

**Navigate to project folder:**

```bash
cd notifications-app
```

**Install dependencies:**

```bash
npm install
```

**Run the app:**

```bash
npm run dev
```
