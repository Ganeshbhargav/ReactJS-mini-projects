# 🍌🍋 Fruit Counter App

A simple and interactive React application that tracks how many mangoes and bananas Bob has eaten. This project demonstrates the use of React state management and event handling in a clean and responsive UI.

---


## 📌 Features

* 🍋 Count mangoes eaten
* 🍌 Count bananas eaten
* 🔄 Real-time UI updates using React state
* 📱 Responsive design (mobile & desktop)
* 🎯 Beginner-friendly React project

---

## 🛠️ Tech Stack

* React JS
* JavaScript (ES6)
* HTML5
* CSS3

---

## 🧠 Concepts Used

* React Functional Components
* `useState` Hook
* Event Handling
* Conditional Rendering (optional enhancement)
* Responsive Design with Media Queries

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/fruit-counter.git
```

2. Navigate to the project folder:

```bash
cd fruit-counter
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```
